Abstract

Smart vehicles will be equiped with a multitude of sensors in order to capture information about its surrounding scene. One of these sensors could be a LIDAR sensor (Light Detection and Ranging) which forms a to-scale point cloud of its surroundings. The goal of this project was to implement the RANSAC (Random Sampling Consensus) algorithm to detect the dominant planes within a point cloud in order further help guide driverless vehicles.

RANSAC Algorithm

The RANSAC algorithm is is an iterative algorithm used for robust estimation of mathematical models from a set of data points that may contain outliers. It was introduced by Fischler and Bolles in 1981 and has been widely adopted in computer vision, image processing, and other fields.

The RANSAC algorithm is particularly useful when the data set includes a significant number of outliers, which are data points that do not conform to the expected model. It aims to find the best-fitting model by iteratively selecting random subsets of data points, fitting a model to each subset, and then evaluating the quality of the model based on a predefined threshold.

Here is how the algorithm is implemented for this project:

1. Initially, no dominant plane has been found, and the best support is set to 0 (see Step 5.)
2. Randomly draw 3 points from the point cloud.
3. Compute the plane equation from these 3 points.
4. Count the number of points that are at a distance less than eps (Îµ) from that plane. This number is the support for the current plane.
5. If the current support is higher than the best support value, then the current plane becomes the current dominant plane and its support is the new best support.
6. Repeat 2 to 5 until we are confident to have found the dominant plane.
7. Remove the points that belong to the dominant plane from the point cloud. Save these points in a new file.
